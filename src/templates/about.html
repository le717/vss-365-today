What is VSS 365?

So you've heard about this thing called VSS 365 and you want to know more about it, such as what it is, how it works, and how to participate. Well, you've come to the right place!

VSS 365 stands for Very Short Story 365. Created by FlashDogs in 2016, every day, participants write a tweet-sized short story based on a daily word prompt. Originally planned to last one year, the prompts are now given out by a new host every month, each with their own unique style and word choices, with the hosts scheduled by Voimaoy.

Getting started

To get started with VSS 365, all you need is a Twitter account. Once the prompt is posted, write a very short story that incorporates the prompt word or uses the prompt as thematic inspiration. Make sure you include the #vss365 hashtag and the prompt word as a hashtag so everyone can find it! That's all there is to it! If you are still wanting more information, Sal Page has written an excellent blog post with further background and writing tips.

About VSS 365 Today

VSS 365 Today is a hobby project created by Caleb Ely, a poet and active participant of VSS 365. This site was created as a way to get the VSS 365 prompt into the hands and minds of more writers more easily, himself included. You can read more about the background of the site on his blog.
