What is #vss365?

Created by FlashDogs in 2016, #vss365 participants write a tweet-sized short story based on a daily word prompt. Originally planned to last one year, the prompts are now given out by a new host every month, each with their own unique style and word choices, with the hosts scheduled by Voimaoy.

Getting started

To get started with #vss365, all you need is a Twitter account. Once the prompt is posted, write a very short story that incorporates the prompt word or uses the prompt as thematic inspiration. Make sure you include the #vss365 hashtag and the prompt word as a hashtag so everyone can find it! That's all there is to it! If you are still wanting more information, Sal Page has written an excellent blog post with further background and writing tips.

What if I don't have Twitter?

If you don't have a Twitter accout, that's perfectly ok! You don't have to be on Twitter to participate. You can still write a story from the prompt, or even use the prompts as inspiration for something else!

What if I miss a day?

Don't worry about that! If you miss a day, you can just that prompt, write multiple stories, or combine the words into one story! There are no hard-and-fast rules here. It's all about having fun with the prompt.

About #vss365 today

#vss365 today is a hobby project created by Caleb Ely, an IT professional, poet, and active participant in #vss365. This site was created as a way to get the #vss365 prompt into the hands and minds of more writers more easily, himself included. You can read more about the background of this site on his blog.

Privacy

Aside from some basic, blockable analytics used to identify browser usage and common referrers to the site, this site does not collect any personally identifiable information. If you voluntarily register for the daily email notifications, your email address is recorded for the sole purpose of sending out the emails through an automatic process and nothing else. Your email can be removed at any time through an unsubscribe link in every email.
